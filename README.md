# nvshen(女神有约)
基于xmppframework 和 openfire的简单聊天系统。
后台系统为java ssh框架缩写，若需要发布表情，请为数据添加支持emoji 
现在后台vps已经到期停用，有兴趣的同学可以自己搭建服务器。
后台代码：https://github.com/hoolang/nvshenJava
